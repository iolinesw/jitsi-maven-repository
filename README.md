<h1>Tutorial para repara&ccedil;&atilde;o de camara com micro na box H96 MAX M9S (8K)&nbsp;</h1>
<p><img src="https://www.powerplanetonline.com/cdnassets/h96_max_m9s_negro_3_ad_l.jpg" alt="" width="330" height="330" /><br /><br /><strong>Guia</strong>-&gt; https://jitsi.github.io/handbook/docs/dev-guide/dev-guide-android-sdk<br /><br /><strong>Depend&ecirc;ncias:</strong><br />Git (<a href="https://git-scm.com/downloads">https://git-scm.com/downloads)</a><br />Android Studio (<a href="https://developer.android.com/studio">https://developer.android.com/studio</a>)<br />Adoptium (<a href="https://adoptium.net/pt-BR/temurin/releases?version=17">https://adoptium.net/pt-BR/temurin/releases?version=17</a>)<br />Java (<a href="https://www.java.com/en/download/manual.jsp">https://www.java.com/en/download/manual.jsp</a>)</p>
<p>Instalar/criar variavel no windows para o ADB para conectar no android da box</p>
<p><br /><strong>Procedimento:</strong><br />&nbsp;<br />1- Fazer <strong>git clone https://github.com/jitsi/jitsi-meet.git</strong><br />2- <strong>cd jitsi-meet/android</strong><br />3- <strong>npm install</strong><br />4- Adicionar num novo ficheiro com o nome "<strong>local.properties</strong>" em <strong>jitsi-meet/android</strong> com o sdk_dir. (ex: sdk.dir=C:\Users\IOLine\AppData\Local\Android\Sdk)<br />5- Ir &agrave; raiz do project e executar: <strong>./android/scripts/release-sdk.sh C:\Users\&lt;USER&gt;\Documents\GitHub\jitsi-maven-repository\release</strong> (isto tem de ir para este sitio para criar as dependencias)<br />6- O aar esta em <strong>C:\Users\&lt;USER&gt;\Documents\GitHub\jitsi-maven-repository\release</strong><br /><br /><br />7- Para corrigir o <strong>erro</strong> de dependencias que est&atilde;o aparecer:</p>
<blockquote>
<p dir="auto">1. Crie um arquivo chamado "<strong>run-packager.bat</strong>" no floder atual com o seguinte conte&uacute;do: <br />Definir o RCT_METRO_PORT-8081 <br />adb reverso tcp:%RCT_METRO_PORT% tcp:%RCT_METRO_PORT% <br />2. Substituir comandoLine("scripts/run-packager.sh") por commandLine("scripts/run-packager.bat") em build.gradle(/android/app/build.gradle); <br />3.Faz react-native run-android;</p>
</blockquote>
<p dir="auto">&nbsp;<br />Substituir comandoLine("scripts/run-packager.sh") em build.gradle(/android/app/build.gradle) <br />Para:</p>
<p><br />&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; // Run the packager<br />&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; //commandLine("scripts/run-packager.sh")<br />&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; if (System.getProperty("os.name").toLowerCase().contains("windows")) {<br />&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; commandLine("scripts/run-packager.bat")<br />&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; } else {<br />&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; commandLine("scripts/run-packager.sh")<br />&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; }<br /><br /><br />8- ir a "<strong>settings.gradle</strong>" em "repositories" adicionar:&nbsp;</p>
<div style="background-color: #1e1f22; color: #bcbec4;">
<pre style="font-family: 'JetBrains Mono',monospace; font-size: 9,8pt;">maven <span style="font-weight: bold;">{<br /></span>    url <span style="color: #6aab73;">"https://github.com/iolinesw/jitsi-maven-repository/raw/main/release"<br /></span><span style="font-weight: bold;">}</span></pre>
</div>
<p>9- ir a "<strong>build.gradle</strong>" em "dependencies" adicionar:</p>
<div style="background-color: #1e1f22; color: #bcbec4;">
<pre style="font-family: 'JetBrains Mono',monospace; font-size: 9,8pt;">implementation (<span style="color: #6aab73;">"org.jitsi.react:jitsi-meet-sdk:0.0.0"</span>) <span style="font-weight: bold;">{ </span><span style="color: #c77dbb;">transitive </span>= <span style="color: #cf8e6d;">true </span><span style="font-weight: bold;">}</span></pre>
</div>
<p><br />10- Ap&oacute;s inicializa&ccedil;&atilde;o da box vai pedir para correr: "<strong>npx react-native start --host 0.0.0.0</strong>" na raiz do projeto</p>
